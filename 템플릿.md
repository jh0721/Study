# 템플릿

- 튜링완전성

1. 함수 템플릿

   - 함수 템플릿 : 함수 형태로 정의된 템플릿

     -> 어떤 자료형으로 호출될지 결정될때 함수생성

   - 템플릿 함수 : 템플릿에 의해 만들어진 함수

   - 함수 템플릿 특수화( call 순서 : 일반함수, 특수화,템플릿함수)

2. 클래스 템플릿
   - 특수화 : 특수한 경우에 대한 재설정
   - 부분 특수화

3. 템플릿 인자

   - 클래스 템플릿은 매개변수에 기본인자(default값) 포함가능
   - 함수 템플릿은 부분 특수화 지원 안됨
