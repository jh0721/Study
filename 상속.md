- 가상함수,순수가상함수,추상클래스

  1. 가상함수 ( virtual )
  2. 순수 가상함수
  3. 추상 클래스

  1. 다중상속: C++ 컴파일러가 실제로 가리키는 객체의 자료형을 기준으로 하는게 아닌, 포인터 변수의 자료형을 기준으로 판단
